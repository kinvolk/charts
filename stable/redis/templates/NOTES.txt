Redis can be accessed via port 6379 on the following DNS name from within your cluster:
{{ template "redis.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local

Note: Password is hardcoded to the following:
```
requirepass="abc123"
masterauth="abc123"
```


1. Run a Redis pod that you can use as a client:

   kubectl run --namespace default {{ .Release.Namespace }} {{ template "redis.fullname" . }} --rm --tty -i \
    --env REDIS_PASSWORD=abc123 \
   --image kinvolk/osb-redis:0.54.0-m -- bash

2. Connect using the Redis CLI:

  redis-cli -h {{ template "redis.fullname" . }} -a abc123
